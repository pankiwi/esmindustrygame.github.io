Introducción

Imprimir mensajes en Mindustry tiene varios propositos utiles, desde mostrar texto con el fin de comunicar algo, hasta para probar nuestro código mostrando información que nos gustaría saber.

Las instrucciones necesarias para imprimir un Mensaje en un Bloque de mensaje son Print y PrintFlush.

Conectando el bloque al procesador

Para poder realizar esta acción primero que todo necesitaremos tener conectado un bloque de mensaje al bloque de procesador que utilizaremos, para hacer esto debes seleccionar el procesador, al hacerlo podrás notar que el rango del procesador se hace visible, clickea en los bloques dentro de ese rango y se conectarán al procesador automáticamente, además, se le asignará un nombre al bloque automaticamente.



En este caso podremos observar que al conectar el bloque de mensaje al procesador se le asignará el nombre message1, con ese mismo nombre haremos referencia al bloque de mensaje en un futuro dentro del bloque de procesador.

Establecer el texto a imprimir

El siguiente paso después de conectar el bloque de mensaje es establecer lo que queremos imprimir, para ello deberemos agregar una nueva instrucción Print y dentro de ella agregar dentro de comillas dobles el texto que nos gustaría imprimir, en este caso a forma de ejemplo he agregado "Hello Mindustry" dentro de la instrucción.



Imprimir el texto

Una vez establecido el texto a imprimir debemos decirle al procesador que escriba esa información dentro del message1, para esto llamaremos a la instrucción PrintFlush. y le daremos como argumento message1.
